# Logistics Shipping Rates

Simple interest calculation script for logistics and shipping rate calculations.

## Description

This repository contains a shell script for calculating simple interest, useful for logistics and shipping rate calculations.

## Features

- Simple interest calculation
- Command-line interface
- Easy to integrate into logistics systems

## Usage

Run the script:

```bash
./simple-interest.sh
```

Or make it executable first:

```bash
chmod +x simple-interest.sh
./simple-interest.sh
```
